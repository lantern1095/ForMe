# 소스코드 설명
```
#include <stdio.h>
```
- 표준 입출력 (printf, scanf) 등을 사용하기 위해 호출하라
```
void print_name()
```
- print_name 의 함수 생성
```
void print_name()
```
- 메인함수 시작
```
print_name();
```
- print_name 함수 호출
